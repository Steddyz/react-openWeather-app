1. CurrentWeather 
- Отображает текущую погоду 
- Показывает температуру, иконку погоды, ветер

DailyForecast
- Отображает прогноз на 7 дней
- Для каждого дня показывает: день недели, иконку, температуру днем/ночью, ветер

LocationSelect
-Поле поиска городов с автодополнением
-При выборе города передает его координаты в App

Loader
-Отображает анимацию загрузки

ErrorMessage
-Показывает сообщение об ошибке и кнопку "Повторить"

App
-Соединеник компонентов

2. Вспомогательные файлы

api/weather
-Содержит все запросы к OpenWeather API
getWeather(lat, lon) – запрос погоды по координатам
searchCities(query) – поиск городов по названию

.env
-Содержит ключ доступа к api

types/weather.d.ts
-Содержит типы для данных о погоде
